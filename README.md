# Slide Puzzle — KOReader plugin

A minimal classic sliding-tile puzzle that runs inside KOReader.

## Features

- Board sizes from **3×3 up to 7×7** (3×3, 4×4, 5×5, 6×6, 7×7).
- **Tap** a tile next to the gap to slide it, or **swipe** in any of the
  four directions to slide the tile on the opposite side of the gap.
- **Move counter** and live **timer** in the header.
- **Best time** and **fewest moves** tracked separately per board size.
- Per-size **auto-save**: you can leave mid-puzzle and resume later, or
  flip the menu switch to always start a fresh puzzle on open.
- Puzzles are always solvable — generated by randomly sliding tiles
  starting from the solved state.
- **In-place tiles are drawn inverted** (solid black with a white digit)
  so you can tell at a glance which pieces are already in their final
  home without needing colour or animations.
- Full-screen UI with minimal chrome: flat rectangles, thick borders,
  centred digits. No animations or flicker on e-ink.

## How to use

1. Copy the `slidepuzzle.koplugin` folder into KOReader's `plugins/`
   directory (or install from the in-app plugin store if available).
2. Restart KOReader.
3. Open **Tools → Slide puzzle → Play**.

### Menu options

- **Play** — open the puzzle.
- **On open: resume / fresh** — toggle whether reopening the plugin
  resumes the last puzzle of the active size, or always shuffles a new
  one.
- **Reset best results** — clear the saved best times and best move
  counts for every size.

### In-game buttons

- **New** — shuffle a new puzzle at the current size.
- **Size** — pick a different NxN size (3..7). Your progress at the old
  size is saved automatically.
- **Stats** — view the best time and fewest moves recorded for every
  size.
- **Close** — leave the puzzle. The timer is paused and folded into the
  saved elapsed time; it resumes next time you open the same size.

## Controls recap

| Input             | Action                                           |
|-------------------|--------------------------------------------------|
| Tap on a tile     | Slide it into the adjacent gap (if neighbouring) |
| Swipe left/right/up/down | Slide the neighbouring tile of the gap    |
| Tap on solved board | Start a new puzzle at the current size         |
| Hardware **Back** | Close the screen                                 |

## Storage

Saved files live in KOReader's settings directory:

- `settings/slidepuzzle.lua` — current state per size, best results,
  active size, preferences.

Nothing else is written or read. Disabling or uninstalling the plugin
leaves these files untouched; delete `slidepuzzle.lua` to wipe stats.
